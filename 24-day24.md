# Turtle - Flower with Circle

The options in Turtle is endless. You can have more fun as you explore. Here are few.

Drawing many circle,

```python
import turtle

turtle.color("red", "yellow")
turtle.begin_fill()

for i in range(18):
    turtle.circle(120)
    turtle.right(20)

turtle.end_fill()
turtle.done()
```
